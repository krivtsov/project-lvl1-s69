## Installation

`sudo npm i -g braingames170220 `

### Start Games

##### "Parity Check"
`brain-even`

##### "Calculator"
`brain-calc`

##### "The greatest common divisor (GCD)"

`brain-gcd`

##### "Balance"
`brain-balance`

##### "Arithmetic Progression"
`brain-progression`

##### "Is it a simple number?"

` brain-prime`
